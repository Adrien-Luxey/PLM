Monde des tris

Ce monde vous donne les outils pour expérimenter les algorithmes de tris. On peut l'utiliser de deux manières différentes : la première est bien sûr d'écrire les algorithmes de tri demandés. Mais on peut aussi se contenter dans un premier temps de lancer la démo de chaque exercice et observer les algorithmes de tri fonctionner. Cela permet de mieux se rendre compte des différences d'efficacité entre eux.

Views available

A contextual menu allows to switch between a state view and a time view. The time view is a bit more complex than the trivial state view but much more informative: it displays the history of the data, allowing to easily debug your program. The time is represented as abscisse in this graph, each curved line representing a given value of the data. When two lines cross, this means that two values were swapped at this time stamp. This view is a bit hairly at the first glance, but reveals very helpful to understand the inner behavior of sorting algorithms.
