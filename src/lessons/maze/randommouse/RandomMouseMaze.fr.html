La souris folle

La journée de votre buggle commence mal. Il n'a pas eu de chance. Il est tombé dans un piège. Aidez le à sortir de ce labyrinthe.

La sortie est représentée par un baggle et vous devez le ramasser afin de vous échapper du labyrinthe.

Nous allons profiter du fait que le labyrinthe ne soit pas trop grand pour écrire l'algorithme le plus bête possible. Cet algorithme repose sur le hasard et est très inefficace.

While the buggle didn't find the path to the escape, it must proceed the following way: pick a random integer between 0 and 2 by using the provided random3() method and make one of the following actions: moving forward if possible, turn left or turn right.

Objectif de cet exercice

This exercise's objective is to write an algorithm allowing the buggle to find its path out of the maze.

N'oubliez pas de faire ramasser le baggle par votre buggle à la fin de votre code.
