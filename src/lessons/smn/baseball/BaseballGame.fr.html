Baseball

Ce monde implémente le jeu du baseball, dans lequel plusieurs joueurs tentent de regagner leur base. Les bases forment un cercle et ont chacune une couleur qui leur est propre. Il y a deux joueurs de chaque couleur pour chaque base, sauf une, qui n'a donc qu'un seul joueur dans son équipe. Il vous est demandé de réordonner les joueurs pour que leur couleur corresponde à celle de la base sur laquelle ils sont. Le seul mouvement autorisé est celui de l'un des deux joueurs de l'une des deux bases adjacentes à la base n'ayant qu'un seul joueur, qui peut donc boucher le trou de cette base et en créer un sur la base qu'il a quittée. Il y a six fonctions qui vous sont fournies : 
void movePlayerOne(int baseSrc)
Déplace le joueur un de la base baseSrc vers le trou. 
void movePlayerTwo(int baseSrc)
Déplace le joueur deux de la base baseSrc vers le trou. 
int[] findMissingPlayer()
Renvoie un tableau d'entier où le premier élément est l'index de la base contenant le trou et le deuxième élément la position du trou sur cette base 
 boolean isBaseSorted( int baseIndex)
Renvoie VRAI ssi tous les joueurs de la base baseSrc sont sur cette même base 
int getAmountOfBases()
Renvoie le nombre de bases dans le jeu. 
 int indexOfBaseWithOnePlayer()
Renvoie l'index de la base qui n'a qu'un seul joueur sur le terrain
