Baseball

This world implements the baseball game, in which several players want to go back to their bases. The bases make a circle and have their own colors. There are two players for one base, except one, which has only one player in its team. You are asked to make the players go on the bases which colors correspond to theirs. The only allowed move is for one of the two players standing on one of the two bases which are directly next to the base with only one player to move and fill the hole, leaving a new one on the base it just left. There are six functions provided : 
void movePlayerOne(int baseSrc)
Move the player one from the base baseSrc to the hole. 
void movePlayerTwo(int baseSrc)
Move the player two from the base baseSrc to the hole. 
int[] findMissingPlayer()
Return an integer array where the first element is the index of the base with the hole, and where the second element is the hole position within this base 
 boolean isBaseSorted( int baseIndex)
Return TRUE if all players of the base baseSrc are home 
int getAmountOfBases()
Return the amount of bases in the field. 
 int indexOfBaseWithOnePlayer()
Return the index of the base which has only one player on the field
