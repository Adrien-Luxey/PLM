Occurrences d'une valeur

L'objectif de cet exercice est de calculer le nombre d'occurrences d'une valeur (17 dans cet exercice) dans un tableau.

Vous devez donc remplir le corps de la méthode occurrences(int[] tab, int lookingFor) qui prend en paramètre le tableau de valeurs entières dans lequel on souhaite compter le nombre d'occurrences de la valeur lookingFor.

Pour calculer le nombre d'occurrences d'une valeur dans un tableau, il faut parcourir l'ensemble des valeurs du tableau, pour chaque valeur, il faut vérifier si cette valeur est égale à la valeur recherchée, et si c'est le cas incrémenter un compteur. Il est donc nécessaire de définir une variable pour ce compteur.
