Valeur maximale

L'objectif de cet exercice est de calculer la valeur maximale d'un ensemble de valeurs contenues dans un tableau.

Vous devez donc remplir le corps de la méthode maximum(int[] tab) qui prend en paramètre le tableau de valeurs entières dont on souhaite calculer le maximum. Cette méthode retourne en résultat la valeur maximale trouvée.

Pour calculer la valeur maximale d'un tableau, il faut parcourir l'ensemble des valeurs du tableau, pour chaque valeur, il faut vérifier si cette valeur est plus grande que la valeur maximale jusqu'à présent trouvée, et si c'est le cas conservée cette valeur. Cela nécessite donc d'une part une variable pour stocker cette valeur, et d'autre part d'initialiser cette variable avec la première valeur du tableau (il est également possible d'initialiser cette variable avec la plus petite valeur entière qu'il est possible de stocker dans une variable de type int. En Java, cette valeur est Integer.MIN_VALUE).
