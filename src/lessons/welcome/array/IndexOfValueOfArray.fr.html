Indice d'une valeur recherchée

L'objectif de cet exercice est de calculer l'indice d'une valeur recherchée (17 dans cet exercice) dans un tableau.

Vous devez donc remplir le corps de la méthode indexOf(int[] tab, int lookingFor) qui prend en paramètre le tableau de valeurs entières dans lequel on souhaite rechercher une valeur. Le second paramètre lookingFor indique la valeur recherchée. Dans le cas où la valeur recherchée ne serait pas présente, la méthode doit retourner la valeur -1.

Pour calculer l'indice d'une valeur recherchée dans un tableau, il faut parcourir l'ensemble des valeurs du tableau, pour chaque valeur, il faut vérifier si cette valeur est égale à la valeur recherchée, et si c'est le cas retournée l'indice de cette valeur. Une fois la valeur trouvée, il n'est pas nécessaire de continuer à parcourir le tableau.

Quelques fonctionnalités des tableaux

Considérons un tableau de valeur entières nommé tableau.

Ce tableau se déclare de la manière suivante :

int[] tableau;

Avant d'utiliser un tableau, il est nécessaire de l'initialiser. Cela se fait généralement de la manière suivante, en précisant la taille souhaitée pour le tableau entre les crochets :

tableau = new int[17];

La déclaration précédente, définit un tableau de 17 valeurs. La première valeur se situe à l'indice 0. Cette valeur se dénote tableau[0].

Il est possible de connaître la taille d'un tableau, en consultant l'attribut length de ce tableau.

tableau.length; // sera égale à 17

Ainsi, la dernière valeur d'un tableau est donnée par : tableau[tableau.length - 1] ou tableau[16] dans le cas de note exemple.
