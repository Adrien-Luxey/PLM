Searching for a given value

The goal of this exercise is to search the cell of a given value, and return its position.

To that extend, you should fill the method indexOf(), which parameters are the array to explore, and the value to search. If the value lookingFor is not in the array tab, the method should return -1.

L'idée de l'algorithme est de parcourir tout le tableau en vérifiant la valeur contenue dans chaque case. S'il s'agit de la valeur cherchée, vous devez renvoyer l'indice de la case actuellement explorée.

Remember that indices begin at 0 and not at 1. So, if there is 3 cells, their indices will be 0, 1 and 2. There would not be any cell numbered 3.

Remember also that the amount of cells in an array can be retrieved using the length attribute. So, if your array is called tab, its size can be retrieved as tab.length. Note that there is no () after length. An attribute is a sort of variable embedded in another object (here, the array).

Donc, la dernière valeur d'un tableau est donnée par tab[tab.length -1].

Remember also that the amount of cells in an array can be retrieved with the len() function. So, if your array is called tab, its size can be retrieved as len(tab).

So, the last value of an array is given by tab[ len(tab) - 1].
