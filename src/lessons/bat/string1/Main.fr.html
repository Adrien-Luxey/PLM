Jouer avec les chaînes

Les chaînes de caractères ("String") sont les plus simples des types de données complexes :) Elles fournissent différentes opérations, comme obtenir la longueur de la chaîne ou une sous-chaîne...

Une très bonne introduction à ce type est disponible ici : http://javabat.com/doc/string.html.
